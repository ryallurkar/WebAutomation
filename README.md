# WebAutomation

### Usage
        $ git clone https://github.com/ryallurkar/WebAutomation.git
        $ cd WebAutomation
        $ mvn clean test

### Reporting
 - [Allure Reporting](http://allure.qatools.ru/)

- Open Terminal > go to project directory > run 


        $ allure serve allure-results
